# Image Folder Guide

Place final website images in these folders:

- `hero`: main hero and landing images
- `about`: treatment room, team, or shop detail images
- `services`: service and treatment images
- `prices`: service menu, pricing, or treatment package images
- `promotions`: promotion artwork or offer images
- `gallery`: general shop/gallery photos
- `rooms`: massage room and private treatment room photos
- `facilities`: spa facilities, amenities, reception, and detail photos
- `team`: staff and therapist photos

The site imports local images from these folders through the files in `src/data`. Add images to the relevant folder, then update or extend the matching data file if needed.
